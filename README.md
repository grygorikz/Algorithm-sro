1 В чем отличие preorder, inorder и postorder?

Разница в порядке посещения узлов:

Preorder NLR:
сначала корень → левое поддерево → правое
Inorder LNR:
левое → корень → правое
Postorder LRN:
левое → правое → корень

2 Какой обход даёт отсортированный результат?

Inorder

Но только если это бинарное дерево поиска BST. Тогда значения выводятся по возрастанию.

3 Почему обходы часто реализуются через рекурсию?

Потому что:

дерево — это рекурсивная структура (каждый узел — это поддерево)
рекурсия естественно повторяет структуру дерева
код получается короче и понятнее

Альтернатива — через стек (итеративно), но это сложнее.
